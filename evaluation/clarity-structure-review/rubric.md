# Clarity and Structure Rubric (1-5)

Score each dimension from 1 (poor) to 5 (excellent).

## Dimensions

1. Logical flow
   - Is the answer organized in a sensible order?
2. Readability
   - Is wording simple, clear, and easy to follow?
3. Formatting consistency
   - Does the response consistently use sections, bullets, and spacing?

## Minimum passing standard

- Average clarity score >= 3.5
- At least 80% of responses have `overall_clarity_score >= 3`

## Reviewer guidance

- Prefer 2 reviewers and average their scores.
- If only 1 reviewer is available, use this rubric consistently for every row.
